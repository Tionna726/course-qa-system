📚 Course QA System 示例课件

这是一个演示课件，展示 course-qa-system 的三大功能模块。

💡 快速开始：
• 选中文字 → 右键 → 选择操作（划线/笔记/提问）
• 按 N 键打开笔记面板
• 按 Q 键打开提问面板

第一章：划线功能演示

请选中这段文字，然后点击右键， 你会看到三个选项：划线高亮、划线+笔记、划线+提问。

划线功能可以帮你标记重点内容， 所有划线都会自动保存到浏览器本地存储中。

第二章：笔记功能

按 N 键可以打开笔记面板。笔记是独立的功能模块， 与提问功能完全分开。

你可以在笔记中记录学习心得、总结要点、或者提出疑问。 笔记支持标题、内容和上下文关联。

第三章：智能提问

按 Q 键可以打开提问面板。如果你选中了文字， 系统会自动将选中的内容作为上下文发送给 AI。

提问功能使用专家视角的 Prompt 设计， 不会用"小白口吻"解释，而是保留专业术语， 帮助学员建立真正的专业认知结构。

附录：思维框架设计方法论

纵向拆解

代码层：具体实现
    ↓
模块层：功能模块
    ↓
系统层：系统架构
    ↓
架构层：设计范式
    ↓
范式层：问题本质

横向关联

看一个概念时，同时问四个问题：

  • 它和什么相邻？ → 找到概念家族
  • 它替代了什么？ → 理解演进逻辑
  • 它依赖什么？ → 理清依赖关系
  • 在别的领域叫什么？ → 建立跨域映射
🎯 核心命题：
顶层思维 = 在发散与收束之间重建可生长的认知结构。
不是站位高、看大局，而是面对问题时能立刻判断： 我现在卡在哪一层？
